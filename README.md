# 여론 렌즈 (Yeoron Lens)

AI Agent 아이디어 공모 — 프로토타입.

**컨셉:** 어떤 이슈·인물이든 물으면 "지금 사람들이 어떻게 *갈려서* 보고 있는지"의 여론 지형을 펼쳐주고, AI는 편을 들지 않고 *심판/해설자*로서 논쟁 구조를 정리하고 사실을 짚어주는 대화형 에이전트. 아고라(커뮤니티)를 한국 여론을 보는 제1플랫폼으로.

- `index.html` — 대화형 버전 (이슈 렌즈 + 인물 렌즈 + 추적)
- `dashboard.html` — 한눈에 보는 대시보드 버전

> 데이터·문구는 전부 컨셉 시연용 더미입니다. 실제 여론 분석 결과가 아닙니다.
